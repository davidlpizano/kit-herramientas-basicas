# 🗂️ GetDataBack Pro (Software de Recuperacion de Datos)

**GetDataBack Pro** (desarrollado por Runtime Software) es una potente utilidad de recuperación de datos diseñada para restaurar archivos perdidos de tus unidades de almacenamiento (HDD, SSD, tarjetas de memoria, USB). Proporciona soluciones robustas para recuperar información incluso si la tabla de particiones, el registro de arranque o las estructuras de los directorios han sido dañadas o perdidas.

## ✨ Funciones Principales

* **Soporte Universal de Archivos:** Compatible con sistemas de Windows (NTFS, FAT, exFAT), Mac (HFS+, APFS) y Linux (EXT).
* **Seguridad de Sólo Lectura:** Su motor opera exclusivamente bajo modo de sólo lectura, asegurando que tus datos originales no sufran daños ni sobrescrituras.
* **Recuperación ante Desastres:** Capaz de lidiar con problemas por ataques de virus, formateos accidentales, problemas de partición, fallos de energía y borrados directos.
* **Interfaz Intuitiva:** Rediseñada para guiar paso a paso en el proceso de recuperación, al alcance de cualquier usuario.

## 🛠️ Requisitos del Sistema

| Requisito | Detalle |
| :--- | :--- |
| **Sistema Operativo** | Windows (32/64-bit) o ejecución bajo entornos WinPE |
| **Almacenamiento** | Pequeño espacio para instalar, pero se requiere un **disco destino distinto** para guardar los archivos recuperados. |
| **Privilegios** | Requiere permisos de administrador para escanear discos a bajo nivel |

## 🚀 Cómo empezar

1. Descarga el software desde el enlace superior incluido en el repositorio.
2. Descomprímelo, ejecuta el instalador y sigue los pasos de instalación.
3. Al abrir GetDataBack, los pasos generales son:
    * **Seleccionar la unidad:** Elige el disco físico donde ocurrió la pérdida de datos.
    * **Escanear:** El programa localizará los sistemas de archivos (este proceso puede llevar un tiempo).
    * **Recuperar:** Selecciona los archivos que necesitas y cópialos a un disco duro seguro diferente al averiado.
