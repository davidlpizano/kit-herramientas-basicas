# 📊 HWiNFO (Software de Información y Diagnóstico de Hardware)

**HWiNFO** (desarrollado por REALiX) es una solución profesional de información y diagnóstico de hardware capaz de realizar un análisis profundo, monitoreo de recursos y generación de informes. Es la herramienta estándar para usuarios que necesitan conocer hasta el último detalle técnico de sus componentes o vigilar temperaturas y voltajes en tiempo real.

## ✨ Funciones Principales

* **Información Detallada:** Análisis exhaustivo de la CPU, GPU, placa base, memoria RAM y almacenamiento con especificaciones técnicas completas.
* **Monitoreo en Tiempo Real:** Supervisión constante de sensores de temperatura, voltajes, velocidades de ventiladores y consumo de energía.
* **Alertas y Registro (Logging):** Configuración de alarmas para evitar sobrecalentamientos y exportación de registros en formatos CSV o XML.
* **Estado de Salud del Sistema:** Predicción de fallos en unidades mediante datos S.M.A.R.T. y detección de *Thermal Throttling*.

## 🚀 Cómo empezar

1. Descarga el software desde los enlaces superiores.
2. Ejecuta el archivo e inicia la aplicación (se recomienda la versión **Portable** para diagnósticos rápidos).
3. Al abrir HWiNFO, selecciona el modo de inicio deseado:
    * **Sensors-only:** Para ver todas las temperaturas, voltajes y frecuencias en tiempo real.
    * **Summary-only:** Para una vista rápida del modelo de procesador, placa y gráfica.
    * **Full Summary:** Para navegar por el árbol completo de hardware del equipo.
4. Si necesitas analizar el rendimiento bajo carga, utiliza el botón de **"Logging Start"** para guardar los datos en un archivo.
